# Company Product Development 1

A React Native restaurant discovery app built as part of a school course project.

## Tech Stack
- Expo SDK 55
- Firebase (Auth, Firestore, Storage)
- React Navigation
- Expo Location
- Leaflet/OpenStreetMap via WebView

## Setup

1. Clone and install
```bash
   git clone https://github.com/yourusername/Company-Product-Development-1.git
   npm install
```

2. Use the `.env.example` file and remove the .example from it, and add your API key to it like so.
```
   FIREBASE_API_KEY=your_key_here
```

3. Start
```bash
   npx expo start
```
